## 3. Strategy : implement everything with toggles → ablate one knob at a time

Rather than testing a 24-config factorial (too expensive), use **staged ablation** :

- **Phase A : implement** everything paper-faithful (T.10 literal). Toggles exposed via config :
  - `n_epochs_pretrain`, `n_epochs_training_high/low`
  - `num_networks` (default 20)
  - `optimizer.name` (default RangerVA if dep present, else ADAMAX with warning)
  - `train_meta_frozen_in_training` (default True = student L969 override)
  - `second_order.hidden_dim` (default 48 = Pasquali 2010, analog Blindsight D.25)
  - `scheduler.step_size=1`, `gamma=0.999`

- **Phase B : validate** paper-faithful config on 500 seeds × 6 settings. Report numbers.

- **Phase C : ablate** each toggle individually (only if Phase B numbers ≠ paper):
  - `meta override : True vs False` during training()
  - `num_networks : 20 vs 30`
  - `n_epochs_pre : 30 vs 60`
  - `optimizer : RangerVA vs ADAMAX`

Each ablation = 1 additional sbatch run, compared to paper-faithful baseline.

### D.28.b — Port `training()` method (≈ 2-3 h)
**Goal :** add `AGLTrainer.training(...)` matching student L904-1035 bit-exact behavior.

**Key design points :**

1. **`train_meta_frozen_in_training` config flag** — controls whether 2nd-order backprops.
   - Default `True` = student L969 override (2nd-order frozen). Forward still runs for logging
     `loss_2`.
   - `False` = alternative behavior (2nd-order also trains). Enables ablation.
   - **No "bug" language**. The flag is a design choice. Document in docstring as
     "empirical-only behavior — paper silent, student freezes 2nd-order."

2. **Grammar-A pattern generation per epoch** : `generate_batch(GrammarType.A, 45 * factor)`.

3. **Cascade / lr / scheduler / loss plumbing** — identical to pre_train, uses separate
   training-phase optimizers (`lr_training_first_order/second_order`).

4. **Per-epoch metrics** : losses_1, losses_2, precision (calculate_metrics inlined).

5. **Does NOT reset first-order weights after** — only pretrain does the reset.

**Signature :**
```python
def training(
    self,
    n_epochs: int,
    *,
    batches: list[TrainingBatch] | None = None,
) -> tuple[np.ndarray, np.ndarray, np.ndarray]:
    """Port of student training() L904. Runs Grammar-A fine-tuning on pretrained networks.

    The `train_meta_frozen_in_training` config toggle determines whether the 2nd-order
    network updates during this phase. Default True = student L969 behaviour.
    """
```

**Verify :** smoke, parity test vs student on small seed.

**DoD :** `training()` added, 1 commit.

### D.28.c — `AGLNetworkPool` 20-copy replication (≈ 1-2 h)
**Goal :** `AGLNetworkPool` class for per-seed network replication.

**API (copied from initial plan, unchanged) :**
```python
@dataclass
class NetworkCell:
    first_order: FirstOrderMLP
    second_order: SecondOrderNetwork
    optim_1: Optimizer
    optim_2: Optimizer
    sched_1: StepLR
    sched_2: StepLR

class AGLNetworkPool:
    def __init__(self, trainer: AGLTrainer, num_networks: int):
        """Deep-copy trainer's post-pretrain networks into num_networks cells."""
    def train_range(self, start: int, end: int, n_epochs: int) -> dict:
        """Train cells [start:end] for n_epochs using AGLTrainer.training()."""
    def evaluate_all(self) -> list[dict]:
        """Evaluate every cell, return per-cell metrics dict list."""
```

**Copy mechanism :** `copy.deepcopy(trainer.first_order.state_dict())` then
`load_state_dict()`. In-memory (not disk round-trip like student).

**Verify :** unit test cell independence.

**DoD :** class + test + 1 commit.
